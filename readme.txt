このゲームはpythonのレトロゲームエンジン「pyxel」で作られた迷路ゲームです。
プレイヤーは沈没した潜水艦を探検し、金塊を発見することを目指します。

<<操作方法>>
上下左右キー：移動
Pキー：床に目印を付けます。ミニマップにも印が表示されます。
Oキー：一定歩数の間、周囲をスキャンしミニマップに結果を反映させます。通常の通路ではないギミックマスがあった場合はマップの色が変わります。
Iキー：プレイヤーの1マス前方にフェンスを設置します。敵の接近を防いだりすることができます。

そのほかの操作が必要な時はゲーム画面のコンソール上に案内が表示されます。

This game is a maze game made with pyxel, a python retro game engine.
Players explore a sunken submarine and try to find GOLD.

<<Controls>>
Up, down, left, right keys: Move
P key: Marks the floor. The mark will also appear on the mini-map.
O key: Scan the surroundings for a certain number of steps and the mini-map will reflect the result. 
       If there is a gimmick square that is not a normal passage, the map will change color.
I key: Places a fence one square in front of the player. This can be used to prevent enemies from approaching.
